# 账号管理

- 用户账号的管理（用户账号的添加、删除与修改）
- 用户口令的管理
- 用户组的管理



## 一、用户账号的管理

##### 1. 添加用户（在root下）

常用命令：

- useradd -m 【用户名】：自动创建该用户的主目录==（实质上是 往一个文件中写用户的信息 [一切皆文件]）==

- cat /etc/passwd：查看创建的用户



##### 2. 删除用户

常用命令：

- userdel -r 【用户名】：删除用户的时候，将他的目录也一并删掉



##### 3. 修改用户

常用命令：

- usermod -d 【目录】【用户名】：修改用户的目录（修改完毕后再查看配置文件）
